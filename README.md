# R Programming Practice Repository

## Overview

This repository contains a collection of **R scripts** focused on core concepts in **data exploration, preprocessing, statistical modeling, and machine learning**. Each script demonstrates a specific technique or model using R. They cover the end-to-end data science workflow from exploration to model evaluation.

## Installation

To run these scripts, you need:

- R (version 4.x or later recommended)
- RStudio (recommended)

Install commonly used packages:
```r
install.packages(c("tidyverse", "caret", "MASS", "rpart", "class", "cluster", "car"))
```
